Sales Data Cleaning Project
This project loads a raw data set and cleans it, leading to an output of a CSV file ready for analysis. Specifically, the script removes leading/trailing spaces, standarizes format, handles missing values and removes negative prices or quantities. 

How to Run the Script
1. Make sure you have Python and Pandas installed.
2. Open Terminal
3. Navigate to project folder in terminal
   cd ism2411-data-cleaning-copilot/src
5. Run the Script
   python data_cleaning.py
7. The clean file will be generated at
   data/processed/sales_data_clean.csv
